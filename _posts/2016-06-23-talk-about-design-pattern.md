#### 活字印刷解读面向对象

|面向对象好处   |类比           |
| ------------- |:-------------:|
| 可维护  |要改，只需更改要改之字 |
| 可复用  |这些字并非用完这次就无用了，完全可以在后来的印刷中重复使用 |
| 可扩展  |若要加字，只需另刻字加入即可 |
| 灵活性好|可竖版可横版，只需要移动字就好了 |
